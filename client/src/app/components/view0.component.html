About

This application is to book an appointment with the blood bank.

A user arrives at the welcome page and clicks on his/her blood type which would bring the user to another page showing the selected blood type as well as the current stock levels as a percentage of targeted stock levels.

The user can then book an appointment with the blood bank.

After the appointment has been booked, the user can then view the appointment details.
